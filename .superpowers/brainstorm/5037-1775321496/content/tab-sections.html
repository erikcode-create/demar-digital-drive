Tab Content — What Goes Where?

Review each tab's contents. Click any to discuss further.

1

Rankings Tab

Keyword table — keyword, current position, change (▲▼—), best position, URL, SERP features
Trend chart — line chart of top 10 keyword positions over 90 days
Search Console — top queries by clicks, impressions, CTR, avg position
Competitors — who ranks for your keywords, their positions vs yours
2

Agent Activity Tab

Last run status — per-phase pass/fail with duration (intelligence ✅ 42s, analysis ✅ 1m12s...)
Run history — calendar heatmap or bar chart showing pass/fail per night over 90 days
Error log — recent failures with error messages, expandable
Agent timing — which agents are slow, trending slower, or timing out
3

Changes Tab

Review pipeline — approved / rejected / revised counts with trend
Recent changes feed — what was changed, which page, when, reviewer verdict + confidence
Pending review — items waiting in the pending directory
Revision history — changes that needed multiple rounds, what feedback was given
4

Site Health Tab

Category scores — grid of score cards (On-Page, Technical, E-E-A-T, CWV, Security, Schema, Links) with trend arrows
Score trends — line chart of each category score over 90 days
Issues list — all current issues sorted by severity (critical → warning → info), expandable with details
Page-level breakdown — per-page scores, click to see that page's issues

Click any tab you want to discuss or change. If they all look right, just say "looks good" in the terminal.
